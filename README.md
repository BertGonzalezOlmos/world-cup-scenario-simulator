# 🌍 AI World Cup Scenario Simulator

Machine Learning project developed in Python to predict FIFA World Cup match outcomes and simulate thousands of championship scenarios using Monte Carlo Simulation.

---

# 📊 Overview

This project combines predictive analytics, feature engineering and Machine Learning to estimate:

- Match outcome probabilities
- Most likely winners
- Championship scenarios
- Tournament simulations

Instead of predicting a single champion with certainty, the model estimates which teams are most likely to become champions after running thousands of tournament simulations.

---

# ⚽ Match Predictions

For every unplayed match the model estimates:

- Home Win Probability
- Draw Probability
- Away Win Probability

Example:

Argentina vs Spain

Argentina Win: 61%

Draw: 18%

Spain Win: 21%

Predicted Winner:
Argentina

---

# 🎲 Monte Carlo Tournament Simulation

The project simulates the tournament thousands of times to estimate championship scenarios.

Example:

France
23.6%

Argentina
20.4%

Brazil
16.1%

England
11.3%

Spain
8.7%

Rather than predicting a guaranteed champion, the model estimates which teams appear most frequently as champions across all simulated tournaments.
